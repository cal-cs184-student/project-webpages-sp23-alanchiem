cs184 Milestone Status Report

What we've done/Preliminary Results:

  • Merged both group’s project 3-1 repos to create a starting point
  • Started implementing volumetric scattering by considering the probability of intersection of ray with fog particles while ray tracing pixels in image.
  • Given absorption coefficient sigma_a and scattering coefficient sigma_s, we generate a random number r and the distance between ray and the first particle would be -log(r)/(sigma_a + sigma-s) and check if this distance is less than the distance to a primitive. If not, we continue bouncing the light off bounding boxes in our illumination computation.

  • Adding new command line arguments for scattering coefficients and bias in phase function
