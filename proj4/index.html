Summary

Rendering Volumetric Scattering

Our goal is to render scenes that contain participating media such as fog, smoke, or dust. To do this, we would need to render with a volumetric scattering effect where rays sometimes scatter before hitting a surface. This involves implementing a more complex version of irradiance, extinction and absorption coefficients, and a phase function, which fully defines the local behavior of light.

Members: Shrihan Agarwal, Alan Chiem, Nathan Dyre, Suhrid Saha

Problem Description

Currently, our pathtracer that we implemented for project 3.1 assumes that light only scatters when it hits a surface. Consequently, the resulting images replicate an environment in which surfaces are embedded in a vacuum. However in reality, impurities and microscopic particles exist within the space between objects. Since even clean air scatters light, it is important for us to implement a solution to this problem if we want to accurately simulate how light behaves.

This project will particularly be challenging in understanding the mathematical model for volumetric scattering. This involves implementing a new radiance equation that uses an extinction coefficient, the sampling for the radiance using Monte-Carlo, a phase function that adapts to different surface materials, and finally a volume rendering function using the aforementioned phase function.

Our general idea on how to solve it is to use an image-based Monte-Carlo Solution that applies Unidirectional Pathtracing.
